## üì± On-Device AI: The Privacy Revolution

The core of Hushh's architecture is **on-device AI processing**:

| Aspect               | Cloud AI          | Hushh On-Device |
| -------------------- | ----------------- | --------------- |
| **Where AI runs**    | Their servers     | Your phone      |
| **Your data goes**   | To the cloud      | Nowhere         |
| **Works offline?**   | No                | Yes, fully      |
| **Who can read it?** | Company + hackers | Only you        |
| **Privacy**          | Trust-based       | Cryptographic   |

### Technical Implementation

- **iOS**: Apple MLX framework, optimized for A-series/M-series chips
- **Android**: Google MediaPipe + Gemma LLM, with GPU/NPU acceleration
- **Storage**: Local SQLite with AES-256-GCM encryption
- **Keys**: Hardware-backed (Keychain/Keystore)
